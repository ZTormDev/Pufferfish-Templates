# Vintage Story Server
This template allows for users to generate a [Vintage Story server](https://www.vintagestory.at/)

## Prerequisites
.NET Runtime 8.0.15 is preinstalled with the server and setup to run Vintage Story

## Recommended way of configuring
I **HIGHLY** recommend using the following [source](https://wiki.vintagestory.at/Server_Config) for configuring other parts of Vintage Story

## This only supports Linux
